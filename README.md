# Cybersecurity Awareness Chatbot - Part 2

## Overview
A Windows Forms GUI chatbot that educates South African citizens about cybersecurity threats.

## Features
- 🎨 Modern GUI with dark theme
- 🎤 Voice greeting at startup
- 🔑 Keyword recognition (passwords, phishing, scams)
- 🎲 Random response generation
- 🧠 Memory to recall user name and interests
- 😊 Sentiment detection for empathetic responses
- 💬 Natural conversation flow with follow-up handling

## How to Run
1. Open the solution in Visual Studio 2022
2. Ensure `greeting.wav` is in `bin/Debug/Resources/`
3. Press F5 to run

## Requirements
- .NET 6.0 or higher
- Windows OS

## Topics Covered
- Password safety tips
- Phishing scam awareness
- Safe browsing habits
- Suspicious link detection
- Privacy protection
